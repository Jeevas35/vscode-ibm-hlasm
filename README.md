# HLASM Highlighting Extension for VS Code

Minimum featured HLASM highlighter for VS Code.

> Tip: Add editor configuration in `user.settings` to highlight continuation column, e.g. `[hlasm] : { "editor.rulers" : [71, 72, 80]},`
